# House Price Prediction ML Project

## Overview
This project predicts house prices using machine learning regression models.

## Dataset
Kaggle Ames Housing Dataset

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- MAE
- R² Score

## Conclusion
Random Forest performed better in predicting house prices.
